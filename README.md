# href-scraper

A ruby gem to get all the hrefs from a single URL

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'href-scraper'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install href-scraper
